HOW TO USE THIS TOOL
F1 Fan Art Studio — Workflow Guide for Ayush, Anagh & Vanshika
★
The Most Important Thing
The prompt this tool generates is a starting point — not a final prompt. It gives you a strong, structured foundation based on the driver's data. You're expected to paste it into Midjourney / DALL-E / Ideogram, see what comes out, and then refine the prompt yourself based on what looks good or needs fixing.
Upload multiple outputs per version — the tool stores them all. When you find the one you like, mark it approved and use the crop tool to extract print-ready elements.
The Workflow — Step by Step
1
Pick Your Edition Mode
At the top of the Prompt Generator, choose between 🏎 Driver Edition (current 2026 grid driver in their current team colours) or ★ Championship Edition (champion drivers wearing #1 with crown motif and championship years).
Championship Edition: Hamilton (7×) · Verstappen (4×) · Alonso (2×) · Norris (1×). Championship years auto-populate.
2
Select Driver + Product, Tweak Fields
Filter by team, then pick the driver. The form auto-fills with data — colours, number, nickname, unique graphic element. Pick T-Shirt or Hoodie.
TEE COLOR
The physical garment — Black, White, Washed Grey, Cream. Separate from design colours.
PRIMARY / SECONDARY
Design colours — team palette used for numbers, streaks, accents. Not the tee colour.
UNIQUE ELEMENT
The circuit layout or helmet graphic identity of this driver's design.
FRONT BADGE / BACK GRAPHIC
Optional extras — auto-filled from driver data. Edit freely. Style direction is baked into the template automatically.
3
Copy → Paste → Generate → Iterate
Hit 📋 COPY and paste the prompt into your AI tool. The generated prompt is a structured starting point — modify it directly as you iterate.
Primary Tools
Artlist AI
Main image gen tool. Paste the full prompt directly. Use the flat lay / product mockup mode if available.
ChatGPT (DALL-E)
Good for iteration — paste the prompt and ask it to adjust specific elements. "Change the back number to be larger" style edits work well.
Tips
→ If AI adds official logos: append "no logos, no brand marks, no badges"
→ For text accuracy in ChatGPT: ask it to "keep all text exactly as written"
→ Try both tools — Artlist tends to be better for flat lay style, ChatGPT better for quick tweaks
4
Save Prompt, Upload Multiple Images
Click ✦ NEW VERSION to save the prompt. Then in the 🏁 DESIGN VAULT, open the design card and upload as many image outputs as you want for that version — all are stored. Each image gets a thumbnail in the gallery.
You can also create a new version directly from the vault modal (+ NEW VER button) without going back to the generator.
5
Extract Print Elements (Crop Tool)
Once you have a good design image, use the ✂ CROP ELEMENTS button to extract individual design sections:
FRONT — mark the left-chest text block and right-chest badge as separate regions
BACK — mark the full back design, hero number, and graphic overlay as separate regions
SLEEVE — optional sleeve detail if present
After cropping:
Crops save to images/crops/ in GitHub, named e.g. ferrari_hamilton_driver_v2_back_1.png — version is part of the filename so crops from different design attempts never clash. The cropped images appear in the vault modal under "Cropped Elements". Run them through remove.bg or Adobe Express to remove the shirt background before sending to print.
✓
Approve & Track
Once a version has at least one image and looks good, open the modal and click ✓ APPROVE. The card turns green. Use the version dropdown to compare attempts and set the best as active.

Quick Tips
→ The UNIQUE ELEMENT field is the identity of each design — the circuit layout or helmet graphic. Don't remove it, but you can rephrase it for different effects.
→ Style direction is built into the template — you don't need to add mood/vibe instructions. Just focus on the driver-specific fields.
→ Upload multiple images per version — different AI outputs, colour experiments, etc. They all live in the same version card.
→ Crop tool saves to images/crops/ — named by team, driver, edition, version, region type, and index (e.g. ferrari_hamilton_driver_v1_front_1.png). Version is included so crops from different design attempts never overwrite each other.
→ Base prompt templates can be edited in the hidden JSON editor (triple-click "DESIGN OPERATIONS v1.0" in the header to unlock).
→ Images above 5MB upload fine but slowly. Resize to under 2MB before uploading for best performance.
